# Registro y Login de Clientes y Administradores (Backend y Frontend)

## Usar Docker Compose

1. **Levantar el contenedor con Docker Compose:**
   ```bash
   docker-compose up -d
   ```
2. **Detener el contenedor:**
   ```bash
   docker-compose stop
   ```
